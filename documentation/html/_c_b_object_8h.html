Detailed Description

Base structure for other structures.

CBObject is the root structure for all other OOP-type structures in cbitcoin. It provides memory mangement through reference counting. Other structures inherit CBObject through the CBOBjectVT structure and the CBOBject structure itself.

Whenever an object is created it must be released and objects must be released for each time they are retained. Failure to abide by this will cause memory leaks.
